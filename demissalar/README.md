
# Fradmin Attest

French employee resignation letter template - modèle de lettre de démission du salarié.


## Documentation (en)

The template has the following arguments:
- ...

The `adresse` should be in the following format:
```typst
adresse: (
  voie: "30 rue du Pont",
  complement: "bâtiment 4", // optional
  code-postal: "75011",
  commune: "Paris",
)
```
